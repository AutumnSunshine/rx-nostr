# rx-nostr

rx-nostr is a library based on [RxJS](https://rxjs.dev/), which allows [Nostr](https://nostr.com/) applications to easily communicate with relays.

## Installation

```sh
npm install rx-nostr
# or yarn add rx-nostr
```
